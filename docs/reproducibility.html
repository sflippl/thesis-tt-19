2.4 Reproducibility

The collected behavioural and neural data is available in the Python package ‘toneworld’, which was created for the purpose of this dissertation and also contains the behavioural and neural analysis pipeline. A small showcase of its functionality can be found in the appendix. The package relied on the packages ‘copy’, ‘numpy’, ‘pandas’ (McKinney, 2010), ‘setuptools’, ‘tqdm’, ‘math’, ‘scipy’ (Jones et al., 2019), ‘bootstrapped’, and ‘plotnine’. Parts of the analysis specific to this dissertation were conducted in R (R Core Team, 2019) using the packages ‘broom’ (Robinson and Hayes, 2019), ‘dplyr’ (Wickham et al., 2019b), ‘feather’ (Wickham, 2019a), ‘ggplot2’ (Wickham et al., 2019a), ‘grid’, ‘lubridate’ (Spinu et al., 2018), ‘magrittr’ (Bache and Wickham, 2014), ‘muStat’ (Wittkowski and Song, 2012), ‘png’ (Urbanek, 2013), ‘reticulate’ (Ushey et al., 2019), ‘stringr’ (Wickham, 2019b), and ‘tidyr’ (Wickham and Henry, 2019). The dissertation itself was written in R using ‘bookdown’ (Xie, 2019a), ‘knitr’ (Xie, 2019b), and ‘rmarkdown’ (Allaire et al., 2019). The package and the code necessary to reproduce the dissertation is available upon request.
