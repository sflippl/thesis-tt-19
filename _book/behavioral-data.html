A.1 Behavioral data

The behavioral data is saved in the subpackage ‘toneworld.behavioraldata’.

import toneworld.behavioraldata as bd
bd.df_behavior.head()
##         mouse         day block  ...  large_gap  imaging  imaging_block

## 510  gtmk443a  2019-05-31     A  ...          0        0            NaN

## [5 rows x 11 columns]

In this dataframe, each row corresponds to one block. One or more blocks constitute a daily session for a particular animal. The columns ‘mouse’ (the animal ID in small letters and without points, e. g. ‘gtmk443a’), ‘day’ (in the format ‘YYYY-MM-DD’), and ‘block’ (represented by large letters) refer to these different levels of detail. Finally, the column ‘behavior’ contains the behavioral data for this day. If only a subset of mice, days, or blocks are of interest, the function ‘bd.subset_behavior()’ can be used to obtain this subset.

example_behavior = tw.get_behavior(
    mouse = 'gtmk444f', 
    day = '2019-07-04',
    block = 'A'
)

It consists of three data frames that describe the presented sounds, the recorded licks, and the presented rewards. These three data frames can be obtained using the methods ‘get_sounds()’, ‘get_licks()’, and ‘get_rewards()’.
